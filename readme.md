
# EVENHUB

SISTEMA GERENCIMANETO DE EVENTOS 


## Installation

Install project with npm

```bash
  git clone https://github.com/wilsonllucena/api
  cd api
  npm install 

```
    
## Running Tests

To run tests, run the following command

```bash
  npm run test:e2e
  npm run test:unit
```


## Tech Stack

**Server:** Node, Express
